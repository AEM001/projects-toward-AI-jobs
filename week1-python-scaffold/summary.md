# 收获
-  第一次自主构建了项目结构，之前都是不知道怎么组织，比较混乱，以后都知道在做项目的时候注意什么了
-  好歹是写了点python的类和对象的代码，虽然不多，但是最核心的收获是多个文件之间接口和功能的相互配合，最终实现一个完整功能。
    - 比如说task定义了类的基本结构，storage提供持久化，service提供服务，CLI负责交互，文件之间的交互是很有收获的
    - 还有文件内部：一个功能是如何从数据出发，使用不同的逻辑实现，这点比较熟悉，在service中的增删查改等功能中也有体现
- 了解了点新知识：如何使用settings进行config管理，如何使用logger,怎么定义log的配置（format,handler,level等）,以及如何在代码中随手添加（logger.info(),logger.error()等）,以及如何定义异常，然后在代码中捕获异常，实现更为稳健的的代码
